# Thinkerer
Breaking thought Bubbles by reducing information biases.

## Setup

Clone this Repo
```
git clone git@github.com:nicolascepeda/thinkerer.git
```

### Frontend

Run the frontend
```
cd app
npm install
npm run
```

### Backend

Run the backend
```
cd api 
pip install -r requirements
python -m uvicorn index:app --reload
```
